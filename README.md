# 🌍 TravelMate Rentals 🚗🏍️

TravelMate Rentals is a web-based platform that allows users to **rent cars and bikes** for travel. The project provides a simple and efficient interface for booking vehicles, exploring options, and managing rental services. Built with **Node.js**, **Express.js**, **MongoDB**, and **EJS**, this app focuses on ease of use and responsiveness.

## 🚀 Features

- 🔐 User Authentication (Login & Signup)
- 🔎 Browse cars and bikes available for rent
- 🗓️ Book vehicles for specific dates
- 📦 View booking history
- 🛠️ Admin dashboard to manage listings
- 📱 Mobile-responsive UI using Bootstrap
- 📩 Contact form for user queries
